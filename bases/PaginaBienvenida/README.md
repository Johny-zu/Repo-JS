# Información del codigo
Se puede ingresar funcinalidad de JS usando <> en html para hacer que sea parte del codigo aunque es mejor o mas bien mas recomendable tenerlo fuera para que se pueda tener mas operabilidad.

## Recomendaciones
Por lo general en las paginas normales se llega a usar los link references de CSS aplicados en el encabezado de la pagina para que la pagina tenga toda la info, en teoria también se puede con JS pero eso suele probocar ciertos errores o a veces no carga de manera correcta lo que hace que de problemas, por lo que se recomienda usar al principio o al fin de las etiquetas <body>
